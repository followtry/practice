面向对象编程
	消息传递:对象之间通过传递消息实现相互通信
		消息传出后必须有个回应
	类:是具有相同属性和行为的对象的集合
类(面向对象思想)的三大特性:封装继承多态
封装:隐藏属性,方法或者实现细节的过程
	eg:宝马车的成本制作工艺对老总不隐藏,对客户隐藏这就是封装
抽象:只关注事物的重要细节,忽略次要细节

当变量写为类中属性时就有初始化值0或false或null,而在函数中单独写是需要自己初始化.

构造器特点(面试题):
	1.无返回类型, 
	2.名称与类名相同
	3.可以有多个构造器
	4.多个构造器由参数列表(参数个数,参数类型,参数顺序)区别.
	5.无,系统会自动生成无操作无参数构造器;有则不自动生成.
	6.构造器在对象实例化时调用.

注意事项:当构造器没有public时,只能在当前包中使用,如果想在不同包中使用,需要将构造器设置为public

this.id...  调用本类的属性和方法
this()\this(int a)  直接调用无参构造器\或含参数构造器,而且只能写在构造器中第一句

重载特点（识别点）：同一类的同名方法，参数列表不同、返回类型无所谓
重写：存在继承关系的两个方法。子类方法与父类方法同名，同参数列表，同返回类型


父类声明子类实例化
重写时，变量实例化哪个类就会显示哪个类中对应的重写方法。
重写时，参数列表不同，参数名相同时，相当于方法的重载而不是重写

父类对象实例化子类对象时，调用方法可以是子类的，但调用属性只能是自己的。

	class A{
		int id=100;
		public void abc(int a){
	
		}
	}
	class B extends A{
		int id=200;
		public void abc(int a){
	
		}
	}

	A a=new B();
	a.abc(10);   //调用子类的重写的方法
	a.id;	//调用的仍然是父类的属性
