	jdk的下载安装

	在java 源文件中类名可以与文件名不相同,但是编译后生成的.class文件名与类名相同,每个类都声称对应的.class文件,因此不必要将多个类定义在一个文件中.public 类必须与文件名同名,而且一个文件中只能有一个public类.
	
	java特性:
		跨平台\安全性\多线程\简单\面向对象\健壮性
	java基本类型:
		byte -128 ~127
		short  -32768~32767
		int 	-2^31~2^31-1
		long 	-2^63~2^63-1
		float  四个字节
		double  8个字节
		将数存为long类型是在数后面加上L
		为float类型赋带小数的值需在数后面加上f,float精确到小数点后六位
		为double类型精
		char 为两个字节
