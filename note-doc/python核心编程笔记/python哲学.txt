1.每个模块中的mian方法都是用来搞测试用的（除了主程序中的main）

2.所有的模块都有能力来执行代码，最高级别的python语句（没有缩进的语句在模块导入时就会执行，不管是不是真的需要）。所以比较安全的办法就是除了真正需要顶级执行的代码外，几乎所有的功能代码都卸载函数中。

3.整数对象和字符串对象是不可变对象，所以python会很高效的缓存，造成认为Python应该创建新对象时，却没有创建新对象的假象。

4.Python 不支持char 和 byte类型

5.Python中整型指代了（short int 和long），因为整型和long密切融合了。
 Python中的float类型是C语言中的double类型，因为考虑到float精度的好处比不上系统的花销。
 十进制浮点数Decimal可以提供任意精度，但需要导入decimal模块
